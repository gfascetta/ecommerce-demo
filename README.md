# Spring boot demo

## API DOC - Swagger 

Go to:
`http://<host>:<port>/api-docs`

Example: http://localhost:8081/api-docs

Go to:
`http://<host>:<port>/swagger-ui/index.html`

Example: http://localhost:8081/swagger-ui/index.html
